\subsection{Detailed Description}
Cube\+Hash interface. Cube\+Hash is a family of functions which differ by their output size; this implementation defines Cube\+Hash for output sizes 224, 256, 384 and 512 bits, with the \char`\"{}standard parameters\char`\"{} (Cube\+Hash16/32 with the Cube\+Hash specification notations).
